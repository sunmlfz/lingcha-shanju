# PRD - 灵茶山居

## 产品概述

**游戏名称：** 灵茶山居  
**类型：** 休闲放置/经营类小游戏  
**平台：** 微信小游戏 / H5 Web  
**引擎：** Cocos Creator 3.8.x + TypeScript  
**目标用户：** 18-35 岁喜欢国风、种植、养成类游戏的用户  

---

## 核心游戏循环

```
种茶 → 采摘 → 炼制 → 品鉴 → 解锁新品种 → 种茶
```

玩家扮演隐居山林的茶道修行者，在灵山仙境中种植各类灵茶，炼制茶饮，通过品茶提升悟道值，解锁更多茶种与场景。

---

## 功能模块设计

### 1. 种植系统（GardenModule）

- **茶圃地块：** 初始 4 格，可扩展至 16 格
- **茶种类型：**
  - 普通茶：绿茶、白茶、红茶、乌龙（基础解锁）
  - 灵品茶：雨前龙井、白毫银针、正山小种（悟道值解锁）
  - 仙品茶：灵芽仙露、千年古树普洱（任务成就解锁）
- **生长阶段：** 播种 → 发芽 → 成长 → 成熟（4 阶段，每阶段有动画）
- **时间机制：** 实时计时，离线补偿（最多 8 小时）
- **施肥/浇水：** 消耗道具加速生长 10%-30%

### 2. 炼茶系统（BrewModule）

- **炼茶台：** 初始 1 台，可扩展至 3 台
- **制茶流程：** 
  - 萎凋（白茶类）→ 杀青 → 揉捻 → 干燥（普通流程）
  - 每步有迷你小游戏（点击时机/滑动控温）
- **品质系统：** 精品率受玩家操作、时节、茶叶品质影响
  - 普通品 / 精品 / 极品（三档，影响茶价值和悟道值）
- **配方解锁：** 通过任务或悟道值解锁拼配茶、花茶配方

### 3. 饮茶系统（SenseModule）

- **品茶体验：** 点击后播放茶汤动画 + 文字意境描述
- **悟道积累：** 每次品茶+悟道值，极品茶 3x 悟道值
- **灵感碎片：** 随机获得，凑齐后触发特殊剧情/解锁
- **茶友拜访：** 每日 1-3 位 NPC 拜访，赠礼换取稀有材料

### 4. 仓库系统（StorageModule）

- **背包容量：** 初始 30 格，升级可扩展
- **物品分类：** 茶叶原料 / 成品茶 / 道具 / 特产
- **出售功能：** 向过路商人出售换取金叶（货币）
- **金叶用途：** 购买种子、扩建茶圃、升级炼茶台、购买装饰

### 5. 任务成就系统（QuestModule）

- **日常任务：** 每日 3 个（种茶/炼茶/品茶各类），完成送道具
- **主线剧情：** 20 章，逐步揭开灵茶山的故事
- **成就体系：** 100+ 成就，涵盖种植量、炼茶次数、品茶种类
- **排行榜：** 悟道值、藏茶量、炼茶数量（好友/全服）

## 界面设计

### 主界面
- 茶圃全景（中国水墨风格）
- 顶部：金叶数量、悟道值进度条
- 底部 Tab：茶圃 / 炼茶台 / 茶室 / 仓库 / 任务

### 视觉风格
- 国风水墨，淡雅青绿色调
- 动态背景：云雾缭绕、落叶飘零（随季节变化）
- 字体：楷体/宋体组合

## 变现设计

- **月卡：** ¥6/月，每日赠送礼包（稀有种子×1 + 道具×3）
- **礼包：** 节日特供礼包（茶具皮肤 + 大量道具）
- **加速券：** 购买加速道具（¥1-¥30 区间）
- **仓库扩容：** 购买扩容方案
